What is the Compensation section's main landing page?

When an admin clicks "Compensation" in the sidebar, what do they see first?

A

Operations Dashboard (recommended)

The landing page is a live ops view — "what needs attention right now?" Stat cards at the top (today's cut-off status, pending payouts, failed jobs, total GSB distributed this week). Below that: a feed of today's daily cut-off results and any failed/pending items needing manual action. Per-distributor lookup is a search box.

B

Distributor List with Compensation Snapshot

The landing page is the full distributor list with compensation columns added: Personal BV, Title, Left Group BV, Right Group BV, Last GSB Earned, Wallet Balance. Paginated, sortable, filterable. Each row links to that distributor's compensation detail page. Aggregate stats appear as a header banner.

Tabbed section: Overview + Sub-pages (recommended)

A tabbed admin section. The "Overview" tab = ops dashboard (stat cards + today's cut-off feed + failed items). Separate sub-pages: Daily Cut-offs | Weekly Payouts | Carry-forwards | Distributor Lookup. Each sub-page is its own route so admins can bookmark "show me all pending payouts" directly.
