Admin → Compensation → Manual Controls
Fallback controls for failed cut-offs, payouts, and carry-forward corrections. Every action is audit-logged and requires a reason.
⚠ These controls affect real money and wallet balances. Every action is permanently audit-logged with your admin ID, a timestamp, the before/after state, and the reason you provide. There is no undo — use the Reverse action if a credit needs to be walked back. When in doubt, use Retry (which is safe and idempotent) before using Manual Credit.
Available actions
🔄

Retry Daily Cut-off

Re-run the 23:59 GSB calculation for a specific distributor and date. Safe to run multiple times — idempotent if already credited.

📊

Recalculate Carry-forward

Re-compute a distributor's slab-1 weaker CF and power-side CF from their full GSB history. Use after a BV reversal mid-cut-off.

💸

Manual GSB Credit

Credit a custom GSB amount to a distributor's wallet. Requires a reference order ID or reason. Use only when retry fails.

↩️

Reverse GSB Credit

Write a debit entry to reverse a specific GSB credit. The distributor's next payout will be reduced by this amount.

⚡

Force Weekly Payout

Trigger the Tuesday payout batch immediately for a single distributor. Only if the batch failed for this distributor.

🔒

Freeze GSB

Block a distributor from receiving GSB without terminating their account. GSB is calculated but held — not credited to wallet.
